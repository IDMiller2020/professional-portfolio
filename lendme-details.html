LendMe Capstone Project for Codeworks, Spring 2021 Cohort:

LendMe was a group capstone project I built along with David Parker, Ethan Boon, and Lane Davis at CodeWorks. We utilized both pair and solo programming and incorporated Agile/Scrum methods for the two week duration of this project. We programed the client side using HTML, CSS, Bootstrap, and Vue. For the server side we used Node.js and Mongoose/MongoDb. The app allows people to post items they are willing to lend out to someone else and lets people browse and request items they would like to borrow. The search function incorportates geolocation capabilities which are available using Mongoose and MongoDb and allows users to search for items withing a specified distance from their current location. The app also allows borrowers to rate lenders and lenders to rate borrowers. Each persons profile page shows their current average rating.

This website was created using the Lonely BootstrapMade template.
